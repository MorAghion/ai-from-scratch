עד עכשיו הבלש שלנו קרא קבצים דרך Python - פתחנו, קראנו, שלחנו ל-API. עכשיו ניתן לסוכן לקרוא את הקבצים **בעצמו**, בלי שנכתוב שורת קוד.

**בחרו את הכלי שלכם** - הנה שלוש דרכים לעשות את זה:

---

**אפשרות א׳: Claude Code (טרמינל)**

אם התקנתם Claude Code, זה הכי פשוט - הוא כבר יודע לקרוא קבצים:

1. פתחו טרמינל בתיקיית detective-ai/
2. הקלידו `claude` כדי להפעיל את Claude Code
3. שלחו לו:

*"You are Benoit Blanc, a brilliant detective. Read all files in the evidence/ folder. For each file, note the source. Then cross-reference the evidence, find contradictions, and rank each suspect 1-10. Only use information from the files - do not invent facts."*

4. צפו - Claude Code קורא כל קובץ בעצמו (תראו את קריאות ה-`Read` בפלט), מצליב, ונותן ניתוח מלא.

---

**אפשרות ב׳: Claude Desktop + Filesystem MCP**

1. התקינו את שרת הקבצים:
`npm install -g @modelcontextprotocol/server-filesystem`

2. פתחו את קובץ ההגדרות של Claude Desktop:
   - **Mac**: `~/Library/Application Support/Claude/claude_desktop_config.json`
   - **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`

3. הוסיפו את הבלוק הזה (שנו את הנתיב לתיקיית detective-ai/ שלכם):

```json
{
  "mcpServers": {
    "detective-files": {
      "command": "npx",
      "args": [
        "-y", "@modelcontextprotocol/server-filesystem",
        "/full/path/to/detective-ai"
      ]
    }
  }
}
```

4. הפעילו מחדש את Claude Desktop
5. פתחו שיחה חדשה ושלחו:

*"You are Benoit Blanc, a brilliant detective. Use the filesystem tools to read all files in the evidence/ folder. Cross-reference the evidence, find contradictions, and rank each suspect 1-10. Only cite facts from the files."*

6. שימו לב - Claude מבקש אישור לקרוא כל קובץ. זה ה-Host (השומר) בפעולה.

---

**אפשרות ג׳: Cursor / VS Code + MCP**

1. פתחו את תיקיית detective-ai/ ב-Cursor (או VS Code עם Copilot)
2. ב-Cursor: Settings → MCP Servers → הוסיפו שרת filesystem עם הנתיב לתיקייה
3. פתחו את ה-Chat ושלחו את אותו פרומפט של בנואה בלאנק
4. הסוכן יקרא את הקבצים דרך MCP - בדיוק כמו באפשרויות הקודמות

---

**מה השתנה?**

השוו את התוצאה ל-detective.py שבניתם בפרק 5:
- **detective.py** - אתם כתבתם את הקוד שקורא קבצים, בונה פרומפט, ושולח ל-API
- **עם MCP** - הסוכן עושה את הכל בעצמו. אותה תוצאה, **אפס קוד**

זה ההבדל בין "USB ייעודי" (סקריפט שבניתם) ל-"USB אוניברסלי" (MCP) - הסוכן לא צריך כלי קריאה מותאם, כי הפרוטוקול מספק את זה.
