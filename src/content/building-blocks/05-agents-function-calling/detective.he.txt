**שלב 1: tools.py - כתבו את הכלים**:

הסוכן צריך שני כלים: אחד שמציג את רשימת הקבצים בתיקיית `evidence/`, ואחד שקורא קובץ ספציפי וּמחזיר את תוכנו.

נסו לכתוב את שתי הפונקציות ב-Python. רמז: `os.listdir()` מחזיר רשימת קבצים, ו-`open()` קורא קובץ.

@@collapse:exercise:הפתרון שלי
```python
import os

def list_evidence_files():
    """List all files in the evidence folder."""
    files = os.listdir("evidence")
    return "\n".join(files)

def read_file(filename):
    """Read the content of a specific file."""
    filepath = os.path.join("evidence", filename)
    with open(filepath, "r") as f:
        return f.read()
```

**שלב 2: tools_schemas.py - כתבו את "התפריט"**:

Function Calling עובד כי אנחנו מספרים למודל אילו כלים קיימים - בפורמט JSON. הנה דוגמה לסכמה של הכלי הראשון:

```python
{
    "name": "list_evidence_files",
    "description": "List all evidence files available in the case folder. Use this first to see what evidence exists.",
    "input_schema": {
        "type": "object",
        "properties": {},
        "required": []
    }
}
```

שימו לב ל-`description` - כאן אנחנו אומרים למודל **מתי** להשתמש בכלי.

עכשיו **כתבו את הסכמה של `read_file`** לפי אותו פורמט. הכלי מקבל פרמטר אחד: `filename` (מסוג string).

**שלב 3: detective.py - הלולאה הסוכנית**:

זה החלק המרכזי. קודם תקראו את הקוד, ואחר כך תענו על שאלות שיעזרו לכם להבין אותו לעומק.

```python
import anthropic
from tools import list_evidence_files, read_file
from tools_schemas import tools

client = anthropic.Anthropic()

SYSTEM_PROMPT = """Role: You are detective Benoit Blanc. You have access to tools that let you read evidence files.

Task: Solve the murder case. Use your tools to examine all available evidence.

Instructions:
1. First, list the available evidence files
2. Read each file and analyze it
3. Cross-reference all sources - find contradictions between witness statements and the forensic report
4. Identify who is lying and why
5. Rank suspects by probability (1-10)"""

def execute_tool(tool_name, tool_input):
    """Route the model's tool request to the right function."""
    if tool_name == "list_evidence_files":
        return list_evidence_files()
    elif tool_name == "read_file":
        return read_file(**tool_input)

def run_agent():
    # Read the crime scene as the initial message
    with open("crime_scene.txt", "r") as f:
        crime_scene = f.read()

    messages = [{"role": "user", "content": f"Here is the crime scene report:\n\n{crime_scene}\n\nInvestigate this case. Use your tools to read all evidence files and solve the murder."}]

    # The Agentic Loop
    for i in range(10):
        response = client.messages.create(
            model="claude-sonnet-4-20250514",
            max_tokens=4096,
            system=SYSTEM_PROMPT,
            tools=tools,
            messages=messages
        )

        # Did the model decide to use a tool?
        if response.stop_reason == "tool_use":
            # Execute each tool the model requested
            tool_results = []
            for block in response.content:
                if block.type == "tool_use":
                    print(f"Detective is using tool: {block.name}")
                    result = execute_tool(block.name, block.input)
                    tool_results.append({
                        "type": "tool_result",
                        "tool_use_id": block.id,
                        "content": result
                    })

            # Send tool results back to the model
            messages.append({"role": "assistant", "content": response.content})
            messages.append({"role": "user", "content": tool_results})

        # Or did the model decide it's done?
        elif response.stop_reason == "end_turn":
            print("\nDetective Blanc's conclusion:\n")
            print(response.content[0].text)
            return

    print("Investigation reached maximum steps.")

run_agent()
```

**לפני שמריצים - תענו לעצמכם**:

**1.** בסיבוב הראשון של הלולאה, המודל מקבל את זירת הפשע + תפריט הכלים. מה לדעתכם הוא יבקש ראשון?

**2.** כשהמודל מחזיר `stop_reason: "tool_use"` - מה הקוד עושה עם התוצאה?

**3.** מה גורם ללולאה **לעצור**? (רמז: יש שני תנאים)

**4.** למה `messages` גדלה בכל סיבוב? מה יקרה אם לא נוסיף את תוצאות הכלים להיסטוריה?

@@collapse:deepDive:התשובות
**1.** הוא כנראה יפעיל `list_evidence_files` - כי ב-description כתבנו "Use this first". אבל גם בלי ההנחייה הזו, בלש שמקבל תיק חדש היה קודם כל בודק מה יש.

**2.** הקוד מפעיל את הכלי (`execute_tool`), לוקח את התוצאה, ושולח אותה **בחזרה למודל** כ-`tool_result`. המודל מקבל את המידע ומחליט מה לעשות הלאה.

**3.** הלולאה עוצרת כש: (א) המודל מחזיר `stop_reason: "end_turn"` - הוא מוכן לענות, או (ב) עברנו 10 סיבובים - הגבלת בטיחות.

**4.** ה-`messages` הן ה"זיכרון" של השיחה. אם לא נוסיף את תוצאות הכלים, המודל לא יידע מה הכלי החזיר - כאילו בלש שליח ששולח מישהו לבדוק ראייה אבל לא מקשיב לתשובה.
