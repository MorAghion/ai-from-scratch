---
term: Function Calling
full: קריאה לפונקציות
definition: מנגנון שמאפשר למודל לייצר בקשה מובנית להפעלת פונקציה חיצונית - הדרך שבה LLM "עושה" דברים בעולם האמיתי.
---
term: JSON
full: JavaScript Object Notation
definition: פורמט טקסט מובנה שמשמש כ"שפה משותפת" בין המודל למערכת - הגשר בין שפה טבעית לקוד.
---
term: Tool Definition
full: הגדרת כלי
definition: התיאור שנותנים למודל על פונקציה זמינה - שם, תיאור, ופרמטרים. כמו תפריט שהמודל בוחר ממנו.
---
term: Structured Output
full: פלט מובנה
definition: היכולת לגרום למודל לייצר תשובה בפורמט מוגדר (כמו JSON) במקום טקסט חופשי.
---
term: API Call
full: קריאת API
definition: בקשה בודדת שנשלחת לשרת המודל. כל "צעד" של סוכן הוא API call נפרד עם כל ההיסטוריה.
---
term: Stateless
full: חסר מצב
definition: המודל לא שומר זיכרון בין קריאות - בכל פעם הוא מקבל את כל השיחה מההתחלה ומחליט מה הצעד הבא.
